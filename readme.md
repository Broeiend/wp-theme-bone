------------
### CSS Architecture

+ **base** (_base.scss) = The defaults, and Normalize.css.
+ **layout** (_layout.scss) = Defining the grid. 
+ **modules** (_modules.scss) = The reusable, modular parts of the design. 

*Note: Css architecture based on SMACSS: http://smacss.com/*
